Automated Password Rotation

Category: Cybersecurity / Software

Overview

This was a side project that I worked on to help automate password rotation for various services using Hudu and Microsoft 365. The goal was to enhance security by regularly updating passwords without manual intervention.

Technologies Used

  • Microsoft Graph API
  • PowerShell
  • Hudu

Key Features

  • First, it pulled the password from Hudu
  • Then, it authenticated with Microsoft
  • It would generate a random password, change the credentials, and then store the new password back in Hudu

What I Learned

This was one of the first projects where I integrated multiple services using their APIs. I learned how to work with the Microsoft Graph API and how to securely handle credentials in PowerShell. Additionally, I gained experience in automating routine security tasks, which is crucial for maintaining a secure IT environment.
