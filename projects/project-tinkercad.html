3D-Modeling with TinkerCAD

Category: Web / Software

Overview

For my capstone assignment in Software Engineering, I had to design a 3D model and utilize a variety of 3D-Printers to produce a physical version of my design. I used TinkerCAD to create the 3D model, which allowed me to experiment with different shapes and structures before finalizing the design for printing.

Technologies Used

  • TinkerCAD
  • Python

Key Features

  • The snake represents the time spent learning python
  • The coffee cup on the table represented learning Java
  • The papers were representative of the hours spent studying
  • All of this encapsulated in a structure that looked like Lakeland

What I Learned

What I learned from this project included gaining hands-on experience with 3D modeling software and understanding the intricacies of 3D printing technology. I faced challenges in ensuring that my design was both functional and aesthetically pleasing, which required multiple iterations and adjustments. This project also enhanced my problem-solving skills as I navigated the technical aspects of 3D printing, such as material selection and printer settings. Overall, it was a valuable experience that combined creativity with technical skills.
