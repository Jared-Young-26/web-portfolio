Lightweight Contact Manager

Category: Software

Overview

This was a final project I did for my Object-Oriented Programming class. The goal was to create a lightweight contact manager application using Java that allowed users to add, view, edit, and delete contacts. The application featured a simple graphical user interface (GUI) built with Java Swing, making it easy for users to navigate and manage their contacts efficiently.

Technologies Used

  • Apache NetBeans IDE
  • Java Programming
  • Object-Oriented Programming

Key Features

  • This was one of my first projects early in my career
  • Basic CRUD application
  • Simple and intuitive GUI using Java Swing

What I Learned

I learned the fundamentals of object-oriented programming, including concepts such as classes, objects, inheritance, and polymorphism. Additionally, I gained experience in designing and implementing a user-friendly GUI using Java Swing. This project also helped me understand the importance of code organization and modularity in software development.
